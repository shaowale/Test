# 采购 Skills 分类体系

## 1. source_type 分类
- agent_skill：有结构化 Skill 包（SKILL.md）
- prompt_template：主要是模板/提示词
- agent_project：Agent/RAG/workflow 项目
- business_system：ERP/SRM/流程系统
- algorithm_project：优化/预测/排程算法项目
- research_reference：文章/方法论参考

## 2. procurement_stages 分类
按 13 段链路：spend_analysis → intake_sow → ... → category_specialist。

## 3. recommended_action 含义
- adopt：可直接吸收为内部原型
- adapt：需结合制度与数据改造
- reference：用于架构/流程参考
- watch：先观察
- discard：不建议投入

## 4. 评分使用建议
- 业务相关性与可复用性优先
- 成熟度与文档质量决定短期落地速度
- license_risk_score 作为上线前闸门
